# Mandelbrot Set

A simple Python project for generating and visualizing the Mandelbrot set.

## Description

This project explores the Mandelbrot set, a famous fractal created by iterating complex numbers.  
It uses Python to compute the set and display the result visually.

## Requirements

Make sure you have Python 3 installed.

Optional recommended packages for visualization:

- `numpy`
- `matplotlib`

## Installation

1. **Clone the repository:**
```bash
   git clone [https://github.com/shadowsid12/Mandelbrot-Set.git](https://github.com/shadowsid12/Mandelbrot-Set.git)
   cd Mandelbrot-Set
```

2. **Create and activate a virtual environment:**

### Windows
```bash
    python -m venv .venv 
    .venv\Scripts\activate
```

3. **Install dependencies:**
```bash
    pip install numpy matplotlib pillow
```

## Usage

Run the project with:
```bash
    python main.py
```

## Project Structure
```text
.
├── main.py # Entry point for the application
└── Mandelbrot.py # Logic for Mandelbrot Set (MS) calculations
```

## Notes

- `main.py` is the entry point.
- `Mandelbrot.py` contains the Mandelbrot-related logic.
